# Pizza-Sales-Dataset
Dataset and Analysis of Pizza Sales Data..
Hey guys, i'm here and very excited to share with you guys my latest project on Pizza sales Analysis. i was able to check data size,(rows, column), inspect column names and data types. Univariate and groupby analysis. with the help of #pandas #matplotlib.pyplot python tools.
I imported 4 tables then used merge to (left join) the tables to make it a complete data set to carry the following tasks.
 What is the total revenue/sales?
 Find the total quantity sold.
 Find the total orders.
 How many pizza types do they sell?
 Find the average price of pizzas.
 What are the peak hours of sales
 Find the total sales made  on each day of the week. Which day of the week is when sales are made the most?
 Find the top 5 bestsalling pizzas.
 Find the sales made on each month. Any trend noticeable?
 Are there pizza types that are not doing well on the menu?
